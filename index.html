Sea of Green Boilerplate

This project is a set of files with which to start your Sea of Green web projects. It includes the baseline Sass files to start a project as well as favicons, logos, and this very homepage (which is a great start to your HTML as well).

Stylesheets

Sea of Green uses Sass for stylesheets. We use a 6-1 architecture based on Sass Guidelines. Currently, we use PrePros for compilation & compression, though we will be moving to a Gulp setup shortly.

Logos & Favicons

In the /assets directory, you will find the logos for Sea of Green, Hands-on Hydro, and the Green Member program. All logos & lockups are available as .pngs with transparency as well as vector formats (.ai & .svg).

The /images contains an ungodly amount of favicons as well as an .svg file that contains all Sea of Green & Green Member logos as SVG symbols.
